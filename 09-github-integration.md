# GitHub Integration

- Run `/install-github-app` and following prompts
- Find workflows in `.github/workflows` folder

- Default GitHub Actions
  - Mention Actions
    - Mention `@claude`
      - Claude code runs
      - Claude codes completes the tasks it was given in the mention
      - Claude code responds to issue or PR
  - Pull Request actions
    - Create a pull request
    - Claude code runs
    - Claude code reviews changes
    - Claude code posts a full report on the imact of the pull request
