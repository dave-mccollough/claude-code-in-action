# Making Changes

- You can paste screenshots into Claude Code using `ctrl + v`

- Planning Mode
- Useful when you have a task that requires a wide understanding of your code base or requires several steps to complete
  - Enable with `shift + tab`
  - Claude will plan and show you the changes it is going to make
  - You can accept them or redirect Claude

- Thinking Modes
- Useful when you are focused on a specific piece of logic that is hard or a specific bug
  - Use specific trigger phrases
    - Less Thinking
      - Think
      - Think More
      - Think a lot
      - Think longer
      - Ultrathink
    - More Thinking
  - Append to your prompt

- You can implent plan and thinking together
- Planning and thiking consume additional tokens
- Claude intergrates with Git and stage and push commits
