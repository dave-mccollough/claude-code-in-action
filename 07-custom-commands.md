# Custom Commands

- Use `/` to view all default Claude commands
- You can create custom commands for automating tasks 
  - In the `.claude` folder create a new directory called `commands`
  - In the `commands` directory, create a `.md` file with your custom command
    - `audit.md` for example 
  - Add instruction to the file
  - Restart Claude code
  - Use the `/` command and search for your custom command
