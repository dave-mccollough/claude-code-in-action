# Controlling Context

- Interrupting or stopping Claude 
  - `Escape` key
  - Interrupts Claude so you can provide alternate instructions

- Repetitive Errors
  - Stop Claude with `Escape` key
  - Add memory with `#` to avoid repative errors

- Review previous messages/conversations with Claude
  - `Escape` + `Escape`

- `/compact`
  - Summarize converstation and continue
  - Helps Claude stay focused but remember what it has learned in current session

- `/clear`
  - Dumps current conversation history
  - Use when switching between tasks
