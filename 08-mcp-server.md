# MCP Servers

- Add new tools to Claude Code using MCP servers
- Can be ran remote or locally
- To add MCP server run the following command 
  - `claude mcp add ....`
  - `claude mcp add playwright npx @playwright/mcp@latest`

- If you want to turn off permission requests
  - Navigate to `settings.local.json`
  - Add MCP server name to allow
